# Leisure Guardian - Android Application


#### 주요 기능
1. 착용자의 상태를 리스트로 확인
2. 지도 화면에서 착용자의 위치 확인
3. 센서 값에 따른 장치의 상태 확인
4. 장치의 등록 및 삭제
